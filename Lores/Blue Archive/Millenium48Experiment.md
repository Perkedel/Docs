# Millenium 48 Experiment

## Ingredients

- Commodore 64 or 64c, Ultimate edition one
- MIDI-SID. Turn Commodore64 Ultimate into arranger keyboard utilizing all 8 SID chips in it.
  - Not to be confused with SID file. SID file is Assembly codes exclusively designed to poke SID chips, [basically MP3 at this point](https://www.lemon64.com/forum/viewtopic.php?t=78350), [you have to have whole Commodore64 itself](https://github.com/UZDoom/UZDoom/issues/883).
  - MIDI-SID Genosify Commodore 64 by working it like SF2. It got presets and the MIDI file again only have to select instruments to begin with. Every instruments are crafted differently on how to poke on SID, dynamically instead of recorded like SID file.
- select students. e.g., Millenium Students
