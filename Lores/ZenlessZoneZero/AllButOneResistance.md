# Resistance `All but one`

An extreme resistancr of an enemy. This enemy is resistant to all kinds of element but one.

So, if a of type of enemy is weak against Ether, then they resistant to everything else. Ice, Fire, Wind, etc., resistant. Except Ether as said. So you must have the ATK and Skill the type of Ether only to make the DMG bonus works well.

It's basically opposite of typical resistance to one type.

## `All no exception`

So far, there is no single enemy found to be resistant to all types, except Test Dummies. So rest assured, until it indeed found, you can enjoy fighting against those Ethereals and Thugs.
