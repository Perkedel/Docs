# Encrypten Script

Encrypten script is inspired from Southeast Asia scripts.

Notably, the vowel ommiter at the end is a *cradle* as a character accent. Download icon but no arrow.

Composing the letter is easy. Top is consonant, and middle is the vowel.

Nowadays, Encrypten script are composed like Korean language or similar, a grid means a syllable. But the original talk stays.

## Language?

Speaking of the talk, Encrypten is like Semarang, where Arenod Sediron is Jakarta of it. Therefore, Encrypten speak is like Javanese.
