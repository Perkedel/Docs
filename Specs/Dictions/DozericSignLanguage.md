# Dozeric Sign Language

Dozeric language is basically Bahasa Indonesia. Like how Arknights call Chinese, Yan.  
The Dozeric Sign Language therefore is based on Indonesian Sign Language that's adapted for intergalactic use, especially English.

Simply learn the gesture for every vocabulary. You can even use simplest form, and the recipent should be able to understand.  
To make it text, speak the text matching the vocab-gesture. each word represents the ID of the vocab-gesture chosen for it. Then whatever sequence of vocab-gestures be translatable to actual sentence of any language. e.g.

A: front!, there's treasure. (Look over there! There's a hidden chest worth looking at.)

B: OK., beware!, possibility trap. (Got it, but be careful tho, it's very obvious looking, must be a trap.)

A: sword, ready-to-use. (Whew almost got me, okay then, weapon's hot.)

B: Forward. (Then we can go)
