# BLDC Washing Machine

Replace that motor with BLDC motor, complete with the ESC of it.

Because LG's idea of disc motor is unrepairable, we're downgrading it back to belted motoring.

But we disagree with the noise brushed motors make, so we use the strongest RC BLDC motor we could find, and use it to replace that legacy unreliable later time motor with it this one. At this point, why not use the one RC motor used for cars and motorcycles since we're doing heave loads?

We can program the ESC e.g. with Bluejay or AM32 depending on the chip architecture, so we can achieve lowest noise profile for heavy load spinings.

Yey! Low noise with repairability!
