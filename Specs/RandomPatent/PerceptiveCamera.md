# Perceptive Camera

This patent might already too mainstream. But Perceptive Camera has lots of use case, and if used by good people, will be useful for peace.

To prove your point, imagine 2 situation of a same scene. The scene is about intersection of a road. Image 1 is car passing back and forth peacefully. And the other is an accident occured because a guilty is running wrong way.  
With Perceptive Camera, the machine can understand if that looks like not normally it is, and then alerted **the operators** who usually are a human. **Perceptive Camera cannot do anything else than to alert the biologics**. It is then up to the operator to escalate this alert.

## How it works

We call Perceptive Camera just a marketing name because the brand focuses only on the goal: **to see, respond, & alert**.  
Building Perceptive Camera to achieve this goal therefore can be done various ways as long as to achive it. e.g., using an AI Model built around the case you'd want the Perceptive Camera works for. Be it this is right and this is wrong, and so on. Teach it how's the usual day should looks like, and which day means wrong.

Perceptive Camera is not only for traffic alert, but also an aid tool that can even give a processed outputs. E.g., Green
