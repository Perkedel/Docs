# Helix Tower

Appartment. Where its hallways are helical ramps, connecting floors inbetween.

You can run in circle to go to different floors instead of lift. In an emergency, if a stair by it is crowded, you can also use this too instead.  
Even better, use small vehicle since the path is a flat surface. weee!!!

Each resident chamber remains untilted. Infact, you got a different options of arrangement based on this design factor, even a Staired connected chambers design where there are 3 sections with each at a different height conforming to the hallway path.
