# Tebenge 2

Sequel to classic now Synthwaved

For Arcade only

2 game modes right now

- Shoot the enemies. Died enemies spawn coins. Every 10 coins in a total score in a session gives 1 ticket.
- Curling. Side this and side that, divide to team. each of member takes turn curling. And when member throw curl hockey, the other jerk their joystick to mop the lane to reduce friction. curls with closest to the target earns most point. members gets ticket according to the points they got. Team this side 20 point in total, then 20 tickets, all members of this side. unused player, rewards will burn.

You can select game mode before enter game. The game mode changes automatically based on schedule, forced schedule even during play, or always manual. It's up to the arcade owner.

Instead of rewards will burn for unused player, by default the reward is set to accumulate. Whoever swipe token card (and it logs you into the card swiper) will automatically claim the ticket. Assignation of ticket give to which token card last depending on that card swipe system. Usually 5 minutes retainment, the interface must tell this game if the logged in card logs out so the game will set ticket mode to accumulation, awaiting for another swipe of whoever token card is. Also must know if account changed too. Basically we only know if the card swiper says logged in or not, no info obtained at all. We spit ticket finishing the accumulation if it says logged in.
