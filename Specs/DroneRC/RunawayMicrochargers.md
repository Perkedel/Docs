# Minicars Runaway

a type of Die Cast. `Runaway` is a RCless motorized Minicar with power based on recharged capacitor. You charge it and it immediately runs. Release and it runs until the capacitor juice runs out.

We lost [Micro Chargers](https://youtu.be/JfuhUQNKJ4U), so we gotta have one again. Not only now Stork Minicars has Hot Wheels size, we of course have micro sized one, replicating that Micro Chargers we lost.

Same Design Type! Bottom Blue for racing, Bottom Red for stunts. Some bottom may only have the color dot instead when its design supposed to be e.g. baremetal bottom.
