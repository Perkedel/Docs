# Space

Space is folder of your Browsing project. It's the upper folder before the actual Tab files inside it. Therefore, your directory will be like

```txt
Tabs
    SpaceMarine
        Website
        Mods
            Download Now
                Not Yet Downloaded 0
            OK
    3D Models
        Get 3D Models from here
            sketchfab.com
        Download the models
            Not Yet Downloaded 0
                Kitbashables PD pack
            OK
                Proteinic | 3D Model Repo

```

You can also load other spaces like loading different coding project folder at a different location, onto this workspace.
